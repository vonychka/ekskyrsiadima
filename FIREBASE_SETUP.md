# Настройка Firebase Service Account для сервера

## 1. Создайте Service Account в Firebase Console

1. Перейдите в [Firebase Console](https://console.firebase.google.com/)
2. Выберите проект "exursional"
3. Нажмите на ⚙️ (Настройки проекта) → "Учетные записи служб"
4. Нажмите "Создать учетную запись службы"
5. Дайте имя: "exursional-server"
6. Выберите роль: "Редактор" или "Администратор Firebase"
7. Нажмите "Создать и продолжить"
8. Пропустите шаг "Предоставить доступ к этой учетной записи службы"
9. Нажмите "Готово"

## 2. Скачайте ключ JSON

1. В списке учетных записей служб найдите созданную
2. Нажмите на нее (email)
3. Перейдите на вкладку "КЛЮЧИ"
4. Нажмите "ДОБАВИТЬ КЛЮЧ" → "Создать новый ключ"
5. Выберите тип "JSON"
6. Нажмите "Создать"
7. Файл ключа скачается автоматически

## 3. Обновите firebase-service-account.json

Замените содержимое файла `firebase-service-account.json` на скачанный ключ.

## 4. Разверните правила Firebase

Выполните команду:
```bash
chmod +x deploy-firebase.sh
./deploy-firebase.sh
```

## 5. Проверьте работу

Перезапустите сервер и проверьте логи - должно быть "Permission denied" ошибок.
